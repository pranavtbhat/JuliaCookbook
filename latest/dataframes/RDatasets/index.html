R Datasets

If you miss the standard datasets from R, or just need a few sample datasets to get familiar with DataFrames, the RDatasets package would be a great place to look.

The list of R packages, from which datasets are available, can be obtained using:

julia> RDatasets.packages()

The datasets available from each package can be obtained by running RDatasets.datasets("PACKAGE_NAME"). For example, for a list of datasets for Econometrics, run:

Or you can simply get a long list of all supported datasets by running:

RDatasets.datasets()

Once you've identified the dataset you wish to use, it can be loaded into a DataFrame by running dataset("PACKAGE_NAME", "DATASET_NAME"). For example, you can load the dataset Income Inequality in the US by running:

ineq = dataset("Ecdat", "incomeInequality")
